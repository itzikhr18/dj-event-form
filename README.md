## מבנה הפרויקט
```
dj-event-form/
├── index.html   ← כל המערכת בקובץ אחד
├── logo.png     ← לוגו הדיג'יי (מוצג בטופס וב-PDF)
└── README.md
```

## איך זה עובד (הזרימה ללקוח)
1. הלקוח ממלא את הטופס (פרטים, שירי מפתח, אווירה, חוקי ברזל).
2. לוחץ "סיום ושליחה לידידיה".
3. נפתח וואטסאפ עם הודעה **מוכנה ומסודרת** שמכילה את כל הפרטים +
   קישורים לחיצים ליוטיוב/ספוטיפיי — ישירות למספר של הדיג'יי.
   נשאר רק ללחוץ "שלח". **אין הורדת קובץ ואין צירוף ידני.**
4. אופציונלי: אפשר להוריד גם עותק **PDF מעוצב** לארכיון.

## העלאה ל-Vercel (הדרך הכי קלה — בלי Git)
1. היכנסו ל-`vercel.com` והתחברו.
2. לחצו על **Add New → Project**.
3. גררו את **כל התיקייה** (Drag & Drop) — חשוב שגם `logo.png` יעלה לצד `index.html`.
4. Vercel מזהה אוטומטית שזה אתר סטטי — לחצו **Deploy**.
5. תוך כ-30 שניות תקבלו כתובת חיה (למשל `dj-event-form.vercel.app`).

> אין צורך בהגדרות build. זה אתר סטטי — Vercel פשוט מגיש את הקבצים.

## העלאה דרך GitHub (אם תרצו ניהול גרסאות בהמשך)
1. צרו ריפו חדש ב-GitHub והעלו אליו את הקבצים.
2. ב-Vercel: **Add New → Project → Import** את הריפו.
3. כל שינוי שתעלו ל-GitHub ייפרס אוטומטית.

## הערות טכניות
- כל המידע נשאר במכשיר של הלקוח. שום דבר לא נשמר בשרת.
- הוואטסאפ הוא ערוץ המסירה הראשי — הודעת טקסט עשירה עם קישורים לחיצים,
  נשלחת ישירות למספר של הדיג'יי (`wa.me`).

  (`enableLinks: true`), ומופק רק בלחיצה על כפתור ההורדה האופציונלי.
