# טופס אירוע — ידידיה יונה (DJ)

טופס צד-לקוח (ללא שרת) קצר ופשוט שאוסף את פרטי האירוע ושולח אותם לדיג'יי
ידידיה יונה בוואטסאפ. המטרה: מילוי מהיר (3–5 דקות) שגם נוח ללקוח וגם נותן
לדיג'יי בדיוק את המידע שהוא צריך — בלי עומס מיותר.

## מבנה הפרויקט
```
dj-event-form/
├── index.html   ← כל המערכת בקובץ אחד
├── logo.png     ← לוגו הדיג'יי (מוצג בטופס וב-PDF)
└── README.md
```

## מבנה הטופס — 4 שלבים
הטופס בנוי כאשף (wizard) של שלב אחד בכל מסך, כדי שירגיש קצר ומהיר במובייל:

1. **פרטי האירוע** — סוג אירוע, שם הלקוח, תאריך, שעת התחלה, כמות אורחים.
2. **מיקום ואנשי קשר** — שם האולם, טלפון, כתובת, קישור ניווט, איש קשר נוסף.
3. **מוזיקה ואווירה** — סגנון מוזיקלי, רמת חופש לדיג'יי, שירי מפתח (לפי סוג
   האירוע), שירי חובה ושירים אסורים.
4. **דגשים אחרונים ושליחה** — שאלה טכנית אחת פתוחה, הערות, ואישור ושליחה.

### שדות חובה (רק הקריטיים)
סוג האירוע · שם הלקוח · טלפון · תאריך · מקום האירוע · שעת התחלה · סגנון
מוזיקלי כללי. כל שאר השדות הם רשות.

### התאמה לפי סוג אירוע
שדות שירי המפתח משתנים אוטומטית:
- **חתונה / חינה** → שיר חופה, שיר שבירת כוס, שיר סלואו.
- **בר / בת מצווה** → שיר כניסה, שיר מרכזי, שירי משפחה.
- **שאר האירועים** → רק סגנון כללי, שירי חובה, שירים אסורים והערות.

## איך זה עובד (הזרימה ללקוח)
1. הלקוח ממלא את 4 השלבים.
2. לוחץ "סיום ושליחה".
3. מופיע מסך סיום קצר עם הודעה ברורה, ונפתח וואטסאפ עם הודעה מוכנה ומסודרת
   למספר של הדיג'יי — **נשאר רק ללחוץ "שלח"**.
4. אופציונלי: אפשר להוריד **PDF מסודר** ולצרף אותו להודעה.

> חשוב: הטופס **לא** נשלח אוטומטית — הלקוח לוחץ על כפתור הוואטסאפ ושולח בעצמו.
> הניסוח בטופס מדויק לכך.

## וואטסאפ ו-PDF
- **הודעת הוואטסאפ** קצרה ומסודרת: פתיח קצר + תקציר הפרטים החשובים.
- **ה-PDF** נקי וקצר, מחולק לאזורים: פרטי לקוח · פרטי אירוע · מיקום ואנשי
  קשר · מוזיקה ואווירה · שירי חובה · שירים אסורים · הערות ודגשים.
- ה-PDF נוצר בדפדפן עם `html2pdf.js` ושומר על קישורים לחיצים.

## העלאה ל-Vercel (בלי Git)
1. היכנסו ל-`vercel.com` והתחברו.
2. **Add New → Project**.
3. גררו את **כל התיקייה** (Drag & Drop) — חשוב שגם `logo.png` יעלה.
4. לחצו **Deploy** — אתר סטטי, ללא הגדרות build.
